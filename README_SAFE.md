# Hyperliquid Explorer

Real-time blockchain explorer for Hyperliquid powered by Lava Network.

## Setup

```bash
npm install
cp .env.example .env.local
# Add your Lava Network RPC URL to .env.local
npm run dev
```

## Tech Stack

- Next.js 14
- TypeScript  
- Tailwind CSS
- TanStack Query
- Lava Network RPC

## Features

- Block explorer
- Transaction search
- Real-time updates
- Network switching (Mainnet/Testnet)
- Dark/Light mode

Built for HyperEVM Hackathon 2025.
